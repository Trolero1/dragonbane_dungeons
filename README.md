# Dragonbane Dungeons 🐉
Este es un proyecto de Python para gestionar mazmorras para un juego de rol,   hecho por un anciano rolero con escasos conocimientos de Python y la ayuda de ias

## 🚀 Funcionalidades
- Editor de pantallas para las mazmorras.
- Persistencia de datos en formato JSON.
- Interfaz gráfica o utilidades de UI.

## 🛠️ Tecnologías utilizadas
- **Python 3.14**
- **pygame-ce**
- **Git & GitHub** para el control de versiones.

## 📁 Estructura del proyecto
- `data/`: Contiene los archivos JSON de las mazmorras.
- `screen_editor.py`: El script principal del editor.
- `.gitignore`: Archivo para excluir archivos temporales de Python.

## ⚙️ Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com
   ```

2. **Entrar en la carpeta:**
   ```bash
   cd dragonbane_dungeons
   ```

3. **Ejecutar el editor:**
   ```bash
   python screen_editor.py
